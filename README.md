# todo-app
A simple Todo Application built using JavaScript, LocalStorage, and an external API. This project demonstrates how to fetch data from a public API, store selected data in the browserâ€™s LocalStorage, and dynamically render and manage todos on the UI without refreshing the page.  The app fetches todos from:  https://jsonplaceholder.typicode.com/todos   Only the first 20 todos are stored locally and displayed. Users can delete any todo or toggle completion status, and all changes are saved instantly in LocalStorage.  This project covers:  Fetching external API data  Storing and retrieving data from LocalStorage  Dynamic DOM rendering  Delete functionality  Status toggle (bonus)  Clean and modular JavaScript functions  Perfect for beginners learning API handling, browser storage, and dynamic UI updates.
